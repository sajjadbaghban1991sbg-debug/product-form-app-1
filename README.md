
# Product Form — Electron + React + SQLite

نسخهٔ آماده برای توسعه. این پروژه یک اپ دسکتاپ الکترون است که:
- فرم با ستون‌های: ردیف، نام محصول، شماره قطار، نوع مسیر، مقدار محصول، کلاس محصول، تاریخ
- مدیریت لیست‌های از پیش‌تعریف‌شده (محصول، قطار، مسیر، کلاس) با افزودن/ویرایش/حذف
- ذخیره‌سازی در SQLite (فایل دیتابیس در مسیر userData ویندوز)
- فیلتر تاریخ و خروجی اکسل

## پیش‌نیازها
- Node.js (نسخه 18+ پیشنهاد می‌شود)
- npm

## راه‌اندازی توسعه
```bash
# داخل پوشه پروژه
npm install
# در یک ترمینال جداگانه برای Dev server
npm run dev
# سپس در ترمینال دیگر
npm start
```

## ساخت فایل نصبی ویندوز (installer .exe)
برای تولید installer نیاز است ابزارهای باینری روی ماشین شما نصب شوند (electron-builder و وابستگی‌های آن). دستورات:
```bash
npm install
npm run build-renderer
# سپس برای ساخت installer ویندوز
npx electron-builder --win
# یا
npm run dist
```
فایل نصبی در پوشه `dist` تولید خواهد شد.

> اگر مایل باشی من پروژه را طوری آماده کنم که تو تنها با یک دستور `npm run dist` خروجی بگیریم (شامل راهنمای نصب NSIS در ویندوز)، داخل فایل راهنما توضیح می‌دهم.

## نکات امنیتی
- دیتابیس در پوشه‌ی کاربری کاربر ذخیره می‌شود.
- برای توزیع داخلی شرکت می‌توانیم رول‌های امضای باینری اضافه کنیم.
